## Implementation Steps

1. Open `packages/draftkit-core/src/serialize.test.ts`.
2. Import `serializeDraftToComponent` from `../serialize` (or from `@draftkit/core`).
3. Write the following `it` blocks inside a `describe("serializeDraftToComponent", ...)` block:

   a. **React import is always first**: call with any valid node; assert `output.startsWith('import React from "react"')`.

   b. **DraftOutput wrapper present**: assert output contains `export default function DraftOutput()`.

   c. **Single component used multiple times**: build a tree with 3 nodes all of type `"Box"`, registry has `Box` with `importPath: "@test/lib"`; assert exactly one `import { Box } from "@test/lib"` line in output.

   d. **Two types from same importPath, sorted alphabetically**: tree has types `"Zebra"` and `"Apple"`, same `importPath`; assert `import { Apple, Zebra } from "@test/lib"`.

   e. **Two types from different importPaths, groups sorted by path**: types `"A"` from `"@z/lib"` and `"B"` from `"@a/lib"`; assert `@a/lib` import line appears before `@z/lib` import line.

   f. **Type with no importPath excluded from imports**: type `"Ghost"` present in tree with no `importPath` in registry; assert no import line for `Ghost`; assert JSX body still contains `Ghost` element.

   g. **Type not in registry excluded from imports**: type `"Unknown"` not present in registry at all; assert no import line for `Unknown`.

   h. **Empty registry**: registry is `{}`; assert only `import React from "react"` is emitted (no named import lines).

   i. **Single leaf node**: one node, no children; assert exactly one named import and a self-contained JSX body.

   j. **Deeply nested tree (5+ levels)**: build a 6-level deep chain of different component types; assert all 6 types appear in the import section.

4. Run `pnpm --filter @draftkit/core test` — all cases pass.
5. Run `pnpm --filter @draftkit/react test` — no regressions.
6. Run `pnpm test` at monorepo root — full suite passes.
7. Run `pnpm --filter @draftkit/core typecheck && pnpm --filter @draftkit/react typecheck` — 0 type errors.
