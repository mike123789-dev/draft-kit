# T-003: Replace prompt textarea with Mini/Small/Medium preset buttons

## Objective

Remove the freeform prompt `<textarea>` and replace it with three preset buttons (Mini, Small, Medium) that each generate a hardcoded draft, populate the editor with its JSX, and immediately render it through the pipeline.

## Scope

- **Changed files**: `packages/draftkit-react/src/DraftKitShell.tsx`
- **Dependencies**: T-001 (pipeline wiring), T-002 (editor state + `editorCode`)

Key changes:
- Remove `prompt` state, `deferredPrompt`, and the `<textarea>` + label
- Remove "Deferred Prompt:" debug line from the sidebar info box
- Define three hardcoded prompts as constants:
  - **Mini**: a single small component (e.g. `"button"`) → 1–2 components
  - **Small**: a simple card (e.g. `"card with title and description"`) → 3–5 components
  - **Medium**: a form or dashboard section (e.g. `"settings form"`) → 6–10 components
- Replace `handleGenerate` with a `handlePreset(prompt: string)` function that:
  1. Calls `createMockDraft(prompt, registry)` + `serializeDraftToJsx`
  2. Sets `editorCode` to the serialized JSX
  3. Immediately calls `validateAndRenderJSX` and stores `pipelineResult`
- Render three buttons "Mini", "Small", "Medium" in place of the textarea
- Remove the old "Generate Draft" button
