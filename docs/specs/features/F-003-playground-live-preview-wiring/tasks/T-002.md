# T-002: Add LiveEditor and Run button

## Objective

Add a syntax-highlighted JSX code editor (`LiveEditor` from `react-live`) to the sidebar, wire a "Run" button that applies the current editor content through `validateAndRenderJSX`, and update "Copy JSX" to copy the editor text.

## Scope

- **Changed files**: `packages/draftkit-react/src/DraftKitShell.tsx`
- **Dependencies**: T-001 (pipeline result state must exist before editor can feed into it)

Key changes:
- Add `editorCode: string` state (tracks editor text, initialized from `jsxCode` after generation)
- Wrap the editor + preview region in a `LiveProvider` with `code={editorCode}` and `disabled` (suppresses react-live's own auto-render)
- Render `<LiveEditor />` in the sidebar below the JSX `<pre>` block (or replacing it)
- Add "Run" button: calls `validateAndRenderJSX(editorCode, registry)`, stores result in `pipelineResult`
- Update `handleCopyCode` to copy `editorCode` instead of `jsxCode`
- Editing `LiveEditor` does NOT trigger preview update — only Run does
